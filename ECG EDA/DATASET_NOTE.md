# Dataset Note

This folder contains the `ECG EDA` project code, notebooks, docs, and experiment artifacts.

The original WESAD raw dataset is not included in this GitHub upload because the source files are too large for a normal GitHub repository.

Excluded from upload:

- `WESAD/`
- `WESAD.zip`

To reproduce the experiments locally:

1. Download the WESAD dataset separately.
2. Place it under `ECG EDA/WESAD/` or update the dataset path when running the scripts.
3. Run `scripts/train_wesad.py` with the local WESAD root path.
